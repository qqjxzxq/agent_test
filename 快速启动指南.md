## 一、获取项目（2分钟）

### 方式A：直接下载
1. 下载项目压缩包
2. 解压到任意目录

## 三、安装依赖（2分钟）

### 方式A：自动安装（推荐）
直接进入**第四步**，启动脚本会自动安装依赖

### 方式B：手动安装
```powershell
# 进入项目目录

# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
.\venv\Scripts\Activate.ps1

# 安装依赖
pip install -r requirements.txt
```

---

## 四、启动应用（1分钟）

运行 python app/main.py

## 五、访问应用（立即）

### 本地访问（你自己）
打开浏览器，访问：
```
http://localhost:8000
```
